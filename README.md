# Electoral Forensics via Large Deviation Theory (選挙フォレンジック・圧力検知ツール)

## Overview (概要)
本リポジトリは、選挙結果、アンケート調査、プラットフォーム上のランキング等において、観測された「得票率（支持率）」が純粋な自由意志の総和（ランダムな揺らぎ）からどれほど逸脱しているかを定量化するPythonツールです。

大偏差原理（Large Deviation Theory）とカルバック・ライブラー情報量（KL Divergence）を用いることで、集団の意思決定に介入した「同調圧力」「組織票」「情報統制（検閲）」の相対的な強度を、物理学的・計算機科学的な限界閾値に基づいて評価します。情報的健康（Informational Health）や認識論的不正義（Epistemic Injustice）を数理的に実証するための解析フレームワークとして機能します。

## Features (主な機能)
* **インタラクティブな異常検知**: 有権者数、候補者数、得票数を入力するだけで、結果の不自然さを即座に判定。
* **KLダイバージェンスに基づく確率計算**: 期待値からの乖離を対数確率（$\log_{10} P$）として高精度に算出。
* **3つの限界閾値による圧力評価**:
  * `Social Limit` ($10^{-9}$): 強い同調圧力・強固な組織票の存在を示唆。
  * `Physical Limit` ($10^{-80}$): 物理的にあり得ない偏り（宇宙の全原子数に基づく限界）。
  * `Computational Limit` ($10^{-308}$): Float64アンダーフロー限界。絶対的な情報統制やデータ改ざんの疑い。
* **CSVレポート出力**: 解析結果を自動でCSVファイル（`election_pressure_analysis.csv`）として保存。

Theoretical Background (理論的背景)統計力学において、マクロな観測量が平衡状態から外れる確率は $P \approx \exp(-N \cdot D_{KL}(q || p))$ と記述されます。本ツールは、得票率がこの限界を超えて偏る現象を、社会空間における特定意見の「ポテンシャルの谷」への落下（アトラクタの介在）として捉えます。確率は Float64 の限界点（$-308$）に至るまで計算され、自然界における「偶然」という説明が数学的に破綻する境界線を明示します。LicenseThis project is licensed under the MIT License - see the LICENSE file for details.
